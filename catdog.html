DEEP DIVE

What the model learned

🔍

EDGE DETECTION

The first conv filters spontaneously learn horizontal, vertical, and diagonal edges — the fundamental vocabulary of shape.

🧱

FEATURE HIERARCHY

Layer 1 finds edges → Layer 2 combines edges into textures (fur, whiskers, snout) → Dense layers reason holistically.

📐

TRANSLATION INVARIANCE

MaxPooling makes the model robust to small shifts — a dog in the top-left is the same dog in the bottom-right.

⚖️

BINARY CROSS-ENTROPY

The loss function penalises wrong confident predictions heavily — pushing the Sigmoid output towards 0 (cat) or 1 (dog).

💡

RELU ACTIVATION

ReLU (max(0,x)) introduces non-linearity for free. Without it, stacking layers would equal a single linear transform.

🎯

SIGMOID OUTPUT

One output neuron: > 0.5 → Dog; < 0.5 → Cat. Distance from 0.5 represents the model's confidence.
